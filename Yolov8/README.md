# CougarAI YOLOv8 Workshop Series

## Overview

This workshop is a 2 part series in a series of 4, where the first 2 went over OpenCV, and how different
Models are trained. The first part of this workshops 2 part series goes over using the base YOLOv8 (no retraining), 
and then retraining YOLO to detect objects of interest. The next part will go over pose estimation and utilizing
it to algorithmically detect punches.
